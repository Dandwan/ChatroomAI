Settings Surface

每日一图与默认风景图池

默认走本地精选风景图包，确保首屏稳定、冷启动快、无外链依赖；需要时再切换到每日一图 API。

默认图池模式

图片来源策略

建议内置一组纯风景、无人物、无商标的高质量图片作为默认池，并把线上 API 做成可选增强项。

默认风景图池

当前原型使用 4 张本地化样图演示轮播。正式落地建议扩展为 12 到 24 张，并按季节与色温分层。

本地包

每日一图 API 配置

这层不绑定某一家供应商，直接支持“请求地址 + 鉴权 + 字段映射”。这样更稳，也方便以后切换。

Provider-Agnostic

建议的数据契约

为了把“设置每日一图 API”做得足够通用，前端最好使用字段路径映射，而不是写死某个平台返回结构。 

{
  "data": {
    "date": "2026-04-29",
    "image": {
      "url": "https://cdn.example.com/daily/2026-04-29.jpg",
      "title": "Morning Above The Ridge",
      "credit": {
        "name": "Studio Name",
        "link": "https://example.com/credit"
      }
    }
  }
}

推荐交互细节

让封面既有审美价值，也不会干扰使用。图片应该是氛围层，而不是主角本身。

冷启动先显示封面
在新对话空白态保留一块完整封面区域，再自然过渡到输入与消息流。
每日只换一次
基于本地日期和缓存判断，避免一天内频繁变化导致首页失去稳定性。
显示来源与摄影师
放在设置或封面信息层内，保持合法与克制，不打断主界面阅读。

设计重点： 用大面积风景图建立情绪入口，用细线、点阵、标签和纵深留白做“精致感”，而不是用厚玻璃和高饱和渐变堆气氛。
