05 Page Set

给真实产品看的页面级演示，而不是一张总览海报。

下面这组原型把你当前产品拆成 5 个核心界面：新对话、对话中、历史抽屉、设置首页、每日封面设置页。 每个页面单独看，方便你逐页提意见。

New Conversation Active Chat Drawer Settings Home Daily Cover Settings
Page 01

新对话空白态

全屏每日风景封面如何进入真实产品，而不破坏当前顶部摘要和输入区节奏。
Page 02

对话中消息流

封面退成摘要卡后，聊天、技能步骤、推理和输入区该怎么保持安静。
Page 03

历史抽屉

保留现有分组和新建/设置动作，但换成更薄更有秩序的版式。
Page 04

设置首页

对照你现在的服务商、模型、Skills、运行时、权限结构做统一视觉重排。
Page 05

每日封面设置页

默认图池、自定义每日一图 API、字段映射和失败回退如何接进真实设置层。
