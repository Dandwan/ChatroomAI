Core rule

消息流里，视觉要让位给阅读。

这页不是“好看就行”，而是要真实服务你的 Skill steps、Reasoning panels、Markdown 内容、图像附件和模型指标。换句话说，聊天态的审美要克制。

Targeted suggestions
封面退化成摘要卡 只保留一张短横幅，用来延续“今日风景”的上下文，不再占满背景。
Assistant 直接读正文 Assistant 回复不再装进方框，改成正文式排版，阅读负担更低。
