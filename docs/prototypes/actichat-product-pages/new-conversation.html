Why this page matters

封面最该服务的是“开始”，不是“过程”。

这页承接你产品不变的核心约束：冷启动一定进入全新对话。既然进入点已经天然是“新的开始”，那最适合把每日风景封面放在这里，而不是对话中途。

Targeted design points
模式切换放到模型列表底部 技能模式和文本模式不再占首页主视觉面积，也不直接塞在模型卡片里，而是并到模型选择列表的最后一段。
保留 3 个首页统计 继续显示 3 个关键摘要，但把它们压成更薄的下沿信息带，让封面仍然是主角。
主题切换能力保留 真实产品不需要把 theme 暴露成首页主按钮，但首页本身必须能适应 theme 变化，所以这里做了 dark / light / system 预览。
字体改成本地可商用 首页不再依赖在线字体。中文标题、中文 UI 和英文斜体都改为可免费商用且可本地托管的字族。
Prototype control

这里不改真实产品逻辑，只调整首页展示结构。日期切换看封面轮换，主题切换看空白态在不同主题里的适应性。
