Why a dedicated page

这不是一个小开关，所以不该埋在“显示选项”里结束。

每日风景封面影响的是产品的进入方式、缓存策略、来源合法性、失败回退和视觉一致性。它值得一个单独详情页。

Targeted suggestions
本地池优先 默认图池是基础设施，不应被 API 成功与否绑架。
通用 API 模型 不要写死某一家，正式实现时支持 endpoint + auth + field mapping。
显示策略要明确 这里把“只在冷启动新对话强展示”讲清楚，避免以后产品行为飘掉。
来源说明低干扰 摄影师和来源信息应该存在，但放在这里和封面信息层里，不打断聊天主流程。
