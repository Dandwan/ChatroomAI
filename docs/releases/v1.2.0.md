# ChatroomAI v1.2.0

## Highlights

- Overhauled transcript storage and prompt settings so conversations, attachments, workspaces, and transcript events have a cleaner persistence model.
- Refreshed the default system prompts and improved skill ZIP installation and mode card behavior.
- Added queued skill-turn appends while a skill request is still running, so users can continue a skill-mode conversation without waiting for the current turn to finish.
- Added a floating "scroll to bottom" control with smoother auto-follow behavior for long conversations.
- Updated workspace info prompts to resolve and expose the absolute workspace path on-device instead of only a relative path.

## Engineering Notes

- Added Android emulator/build notes for this machine in [docs/android-emulator-skill-update.md](../android-emulator-skill-update.md).
- Updated lint ignores so repo-local Gradle caches do not pollute release verification.
