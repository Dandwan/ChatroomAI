# `cloud-server/admin-ui/src/styles/admin.css`

## 功能
Admin UI 的全局样式表。定义 CSS 自定义属性（暗色主题）、布局（侧边导航 + 主内容区）、表单元素、表格、徽章、卡片、图表容器、登录页等样式。**v5: 新增 `.admin-key-code`（Key 代码展示）、`.admin-key-ft`（Key 容错输入行）、`.admin-input-xs`（超小输入框）样式，支持 per-key 容错配置 UI。**
